# 🚀 Cours PHP Complet - De Débutant à Avancé

## 📋 Description

Ce cours PHP complet vous permettra de maîtriser le langage PHP depuis les bases jusqu'aux concepts avancés. Chaque chapitre est conçu pour être pratique avec de nombreux exemples de code.

## 🎯 Objectifs

À la fin de ce cours, vous saurez :
- ✅ Maîtriser la syntaxe PHP et les concepts fondamentaux
- ✅ Programmer en orienté objet avec PHP
- ✅ Gérer les bases de données avec PDO
- ✅ Créer des applications web sécurisées
- ✅ Appliquer les bonnes pratiques de développement

## 📚 Contenu du cours

### 🔰 **Niveau Débutant**

#### **Chapitre 1: Introduction à PHP**
- Qu'est-ce que PHP ?
- Installation et configuration
- Syntaxe de base et balises
- Variables et constantes
- Types de données
- Affichage et commentaires

#### **Chapitre 2: Opérateurs**
- Opérateurs arithmétiques
- Opérateurs de comparaison
- Opérateurs logiques
- Opérateurs d'assignation
- Opérateurs spéciaux (ternaire, null coalescing)

#### **Chapitre 3: Structures de contrôle**
- Conditions (if, else, elseif, switch)
- Boucles (for, while, do-while, foreach)
- Break et continue
- Match (PHP 8+)
- Structures alternatives

### 🔧 **Niveau Intermédiaire**

#### **Chapitre 4: Fonctions**
- Déclaration et appel de fonctions
- Paramètres et valeurs de retour
- Paramètres par défaut et variables
- Fonctions anonymes et closures
- Fonctions fléchées (PHP 7.4+)
- Fonctions récursives
- Portée des variables (scope)

#### **Chapitre 5: Tableaux**
- Tableaux indexés et associatifs
- Tableaux multidimensionnels
- Parcours de tableaux
- Fonctions de tableaux utiles
- Tri et recherche
- Filtrage et transformation
- Fusion et intersection

#### **Chapitre 6: Chaînes de caractères**
- Création et concaténation
- Recherche et extraction
- Remplacement et transformation
- Formatage et nettoyage
- Expressions régulières (PREG)
- Encodage et sécurité

### 🏗️ **Niveau Avancé**

#### **Chapitre 7: Programmation Orientée Objet**
- Classes et objets
- Propriétés et méthodes
- Visibilité (public, private, protected)
- Constructeur et destructeur
- Héritage et polymorphisme
- Méthodes et propriétés statiques
- Constantes de classe
- Méthodes magiques
- Interfaces
- Classes abstraites
- Traits
- Namespaces
- Autoloading
- Exceptions

#### **Chapitre 8: Gestion des fichiers**
- Lecture et écriture de fichiers
- Modes d'ouverture
- Informations sur les fichiers
- Gestion des dossiers
- Copie, déplacement, suppression
- Upload de fichiers
- Formats CSV et JSON
- Sécurité des fichiers

#### **Chapitre 9: Base de données (PDO)**
- Introduction à PDO
- Connexion à la base de données
- Création et gestion des tables
- Opérations CRUD (Create, Read, Update, Delete)
- Requêtes préparées
- Jointures et requêtes complexes
- Transactions
- Gestion d'erreurs
- Sécurité et injection SQL
- Pagination

#### **Chapitre 10: Sessions et Cookies**
- Introduction aux sessions
- Gestion des sessions
- Stockage de données en session
- Destruction de sessions
- Cookies
- Sécurité des sessions
- Authentification
- Gestion du timeout
- Messages flash
- Bonnes pratiques

## 🚀 Comment utiliser ce cours

1. **Commencez par l'index** : Ouvrez `index.php` pour avoir une vue d'ensemble
2. **Suivez l'ordre** : Les chapitres sont conçus pour être suivis dans l'ordre
3. **Pratiquez** : Chaque chapitre contient des exemples pratiques
4. **Expérimentez** : Modifiez le code pour mieux comprendre
5. **Appliquez** : Créez vos propres projets avec les concepts appris

## 📁 Structure des fichiers

```
cours-php/
├── index.php                 # Page d'accueil du cours
├── 01-introduction.php       # Chapitre 1: Introduction
├── 02-operateurs.php         # Chapitre 2: Opérateurs
├── 03-structures-controle.php # Chapitre 3: Structures de contrôle
├── 04-fonctions.php          # Chapitre 4: Fonctions
├── 05-tableaux.php           # Chapitre 5: Tableaux
├── 06-chaines.php            # Chapitre 6: Chaînes de caractères
├── 07-poo.php                # Chapitre 7: POO
├── 08-fichiers.php           # Chapitre 8: Gestion des fichiers
├── 09-base-donnees.php       # Chapitre 9: Base de données
├── 10-sessions-cookies.php   # Chapitre 10: Sessions et cookies
└── README.md                 # Ce fichier
```

## ⚙️ Prérequis

- **Serveur web** avec PHP 7.4+ (XAMPP, WAMP, MAMP, ou serveur local)
- **Base de données MySQL** (pour le chapitre 9)
- **Navigateur web** moderne
- **Éditeur de code** (VS Code, PhpStorm, Sublime Text, etc.)

## 🔧 Installation

1. **Clonez ou téléchargez** les fichiers du cours
2. **Placez les fichiers** dans votre dossier web (htdocs, www, etc.)
3. **Démarrez votre serveur** web et MySQL
4. **Créez une base de données** nommée `cours_php` pour le chapitre 9
5. **Ouvrez** `index.php` dans votre navigateur
